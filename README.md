# LinkedIn Roasted

LinkedIn Wrapped, but it's mean. (Affectionately.)

This is a Next.js project that analyzes your LinkedIn data export and generates a humorous "roast" of your LinkedIn activity.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- Next.js 16 with App Router
- TypeScript
- Tailwind CSS
- Framer Motion (animations)
- JSZip (for LinkedIn export parsing)
- PapaParse (CSV parsing)
- html-to-image (for sharing)
